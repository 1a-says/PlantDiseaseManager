# 智能防控系统

## 项目简介

智能防控系统是一个基于Web的农业病虫害防控管理平台，提供化学精准施药和绿色防控设备管理两大核心功能。

## 功能特性

### 1. 化学精准施药方案生成
- **智能诊断**: 上传病虫害图片，AI自动识别病虫害类型和严重程度
- **方案生成**: 基于诊断结果自动生成专业的防治处方
- **多方案选择**: 提供多种备选防治方案，包括有机和化学防治
- **可视化展示**: 直观的病情展示和处方单界面

### 2. 绿色防控设备一键启停
- **设备地图**: 2D地块设备分布图，实时显示设备状态
- **状态监控**: 实时监控设备运行状态、电量、诱捕数量等
- **批量控制**: 支持分区一键启停设备
- **故障告警**: 设备故障自动告警和状态提示

## 技术特点

- **响应式设计**: 适配桌面端和移动端设备
- **实时更新**: WebSocket实现设备状态实时同步
- **模块化架构**: 清晰的代码结构，易于维护和扩展
- **用户友好**: 简洁直观的界面设计，降低使用门槛

## 使用方法

1. 打开 `index.html` 文件
2. 选择功能模块：
   - **精准施药**: 上传病虫害图片，获取防治方案
   - **绿色防控**: 查看设备状态，控制设备启停
